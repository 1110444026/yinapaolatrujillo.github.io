# AA4-EV01 - Espacio de divulgación

Este paquete contiene un sitio web listo para publicar como evidencia de la actividad 3.4.

## Contenido
- `index.html`: página principal del espacio de divulgación.
- `assets/`: imágenes del recurso educativo digital final.

## Publicación sugerida
Este sitio está preparado para subirlo a una plataforma gratuita como GitHub Pages.

## Pasos básicos
1. Cree una cuenta en GitHub.
2. Cree un repositorio nuevo.
3. Suba el archivo `index.html` y la carpeta `assets`.
4. Active GitHub Pages en la configuración del repositorio.
5. Copie el enlace público generado y súbalo como evidencia.
